Project Overview

A modern Login & Signup System built using HTML, CSS, and JavaScript, focusing on clean UI design and client-side form validation. The project allows users to create an account and securely log in using stored credentials. It demonstrates form handling, input validation, password visibility toggling, and basic authentication logic using LocalStorage.

Key Features

  • User-friendly Login and Signup forms with smooth toggle functionality.
  • Client-side validation for name, email format, and password length.
  • Prevents login without prior signup.
  • Stores user credentials securely in LocalStorage (frontend demo).
  • Show/Hide password option for better.
  • Displays clear error and success messages
  • Responsive layout suitable for mobile, tablet, and desktop devices.
  • Clean and accessible UI with proper form labels and feedback.
